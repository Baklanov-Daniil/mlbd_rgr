# Классификация типа вина по химическому составу

Веб-приложение для инференса моделей машинного обучения, позволяющее определить тип вина (красное/белое) на основе его физико-химических характеристик.

---

## О проекте

Данное приложение разработано в рамках расчетно-графической работы по дисциплине **«Машинное обучение и большие данные»**. Основная цель — демонстрация работы шести различных моделей машинного обучения через удобный веб-интерфейс.

**Задача:** Бинарная классификация — определение типа вина (Red / White) по 12 химическим признакам.

---

## Признаки датасета

| Признак | Описание | Единицы измерения |
|---------|----------|-------------------|
| fixed acidity | Фиксированная (титруемая) кислотность | г/дм³ |
| volatile acidity | Летучая кислотность | г/дм³ |
| citric acid | Лимонная кислота | г/дм³ |
| residual sugar | Остаточный сахар | г/дм³ |
| chlorides | Хлориды | г/дм³ |
| free sulfur dioxide | Свободный диоксид серы SO₂ | мг/дм³ |
| total sulfur dioxide | Общий диоксид серы SO₂ | мг/дм³ |
| density | Плотность | г/см³ |
| pH | Водородный показатель | pH |
| sulphates | Сульфаты | г/дм³ |
| alcohol | Содержание алкоголя | % |
| quality | Оценка качества | балл (0–10) |

**Целевая переменная:** `type` — тип вина (0 = White / белое, 1 = Red / красное)

---

## Используемые модели

| Модель | Описание |
|--------|----------|
| **ML1** | Логистическая регрессия (Logistic Regression) |
| **ML2** | Градиентный бустинг (Gradient Boosting) |
| **ML3** | CatBoost (продвинутый градиентный бустинг) |
| **ML4** | Бэггинг (BaggingClassifier) |
| **ML5** | Стэкинг (StackingClassifier) |
| **ML6** | Полносвязная нейронная сеть (Keras/TensorFlow) |

## Технологии

- **Python 3.10** — основной язык программирования
- **Streamlit** — веб-фреймворк для создания дашбордов
- **Scikit-learn** — классические и ансамблевые модели ML
- **CatBoost** — градиентный бустинг
- **TensorFlow / Keras** — нейронная сеть

- **Matplotlib, Seaborn** — визуализация
